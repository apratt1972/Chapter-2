Description List

TCP
Transmission Control Protocol is a method (protocol) used along with the Internet Protocol (IP) to send data in the form of message units, called packets, between computers over the Internet.
IP
Internet Protocol is the method of protocol by which data is sent from one computer to another on the Internet. Each computer on the Internet is identified by an IP address.
FTP
File Transfer Protocol is a protocol used to exchange files between computers on the Internet.
HTTP
Hypertext Transfer Protocol is the protocol used for exhanging text, graphic images, sound, video, and other multimedia files on the Web.
